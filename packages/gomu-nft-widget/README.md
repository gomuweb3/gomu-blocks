## Getting Started

This project requires first version of yarn.

```bash
yarn set version 1.22.1
```

To install packages:

```bash
yarn
```

To run dev server:

```bash
yarn start
```

Supported env variables (use `.env` file in the root of the project):

```bash
REACT_APP_OPENSEA_API_KEY=
REACT_APP_GOMU_API_KEY=
```
